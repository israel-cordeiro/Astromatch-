### Projeto Astromatch
-Uma rede de relacionamento para Todes.

### Tecnologias:
-ReactJS
-Styled Components
-Axios

### Funcionalidades
- O usuário escolhe entre uma lista de participantes para dar 'match' ou não.

- Os matchs dos usários ficam na salvos, e podem ser deletados caso o usuário queira.
